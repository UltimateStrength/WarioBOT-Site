# WarioBOT — Site

Site estático do WarioBOT, um bot para Discord.

## Estrutura

```
WarioBOT-Site/
├── index.html       # Página principal
├── css/
│   └── style.css    # Estilos
├── assets/          # Imagens (logo, botões, ilustrações)
├── LICENSE
└── README.md
```

## Como visualizar

Abra o `index.html` diretamente no navegador, ou acesse pelo GitHub Pages:  
https://ultimatestrength.github.io/WarioBOT-Site/
